# Код 
---
```c++
class X{
public:
	static constexpr double X_PI{3.14};
	int b{}, a = 1; // данные
	enum {e1, e2, e3} g;
	
	X(): b{1}, g(X::e1) {  // конструктор
		//тело конструктора
	}
	~X() {};//деструктор
	
	void foo(); //метод

private:
	int data_{}; //приватные данные
	
	struct inner {
		int i{}, j{};
		void g();
	};
	inner c;
}

int main()
{
	X x{};
	double d = X::X_PI; 
	x.a = 1;
	x.g = X::e2;
}
```
---
# 1. Класс X и его члены
## Статические члены
---
```cpp
static constexpr double X_PI{3.14};
```
---
- **static** - член принадлежит самому **классу!!**, а не объектам
- **constexpr** - вычисляется на этапе **компиляции**
- **double X_PI{3.14}** - инициализация с помощью **_uniform initialization_** (C++11)
- Доступ: `X::X_PI`

## Поля данных
---
```cpp
int b{}, a = 1;  // данные
enum {e1, e2, e3} g;
```
---
- **int b{}** - **_value-initialization_**, b будет равно 0
- **int a = 1** - инициализация значением 1 (*копирующая инициализация*)
- **enum {e1, e2, e3} g** - анонимное перечисление (enum), переменная g типа этого перечисления

## Приватные члены
---
```cpp
private:
    int data_{};
    struct inner { ... } c;
```
---
- **data_{}** - приватное поле, инициализировано 0
	- Через `_` показывают, что *данные* приватные
- **struct inner** - вложенная структура, доступная только внутри X
- **inner c** - объект вложенной структуры

# 2. Конструктор и деструктор
---
```cpp
X(): b{1}, g(X::e1) {  // конструктор
    // тело конструктора
}
```
---
- **X()** - конструктор по умолчанию
- **b{1}** - инициализация b значением 1 (переопределяет {})
- **g(X::e1)** - инициализация перечислением e1
- Тело конструктора пустое
---
```cpp
~X() {};  // деструктор
```
---
- **Деструктор**, вызывается при уничтожении объекта
- Пустой, так как нет динамической памяти

# 3. Методы и вложенная структура
---
```cpp
void foo();  // метод
```
- Объявление метода
---
```cpp
struct inner {
    int i{}, j{};
    void g();
};
```
- **i{}, j{}** - инициализированы 0
- **void g()** - объявлен, но не определен
---
## 4. Функция main
---
```cpp
int main()
{
    X x{};                 // 1
    double d = X::X_PI;    // 2
    x.a = 1;              // 3
    x.g = X::e2;          // 4
}
```
---
**1. `X x{};`**
- Создание объекта x класса X
- Вызывается конструктор X()
- b становится 1 (из конструктора)
- a = 1 (из инициализации в классе)
- g = X::e1 (из конструктора)
- data_ = 0 (из инициализации в классе)
- c.i = 0, c.j = 0 (из инициализации inner)

**2. `double d = X::X_PI;`**
- Доступ к статическому члену через **класс!!!** (не объект)

**3. `x.a = 1;`**
- Изменение публичного поля a
- Несмотря на то, что a уже было 1

**4. `x.g = X::e2;`**
- Изменение значения перечисления на e2
